# CS_WebPage

## Description
This project is a web page built using React and Vite for the frontend and Node.js for the backend.  


## Progroms Used
- React
- Vite
- JavaScript
- HTML
- CSS
- Node.js
- npm


## How to Run the Project

1. Clone the repository

git clone https://github.com/fv201/CS_WebPage.git

2. Move into the project folder

    cd CS_WebPage

3. Install dependencies

    npm install

4. Start the frontend development server

    npm run dev

5. Start the backend server

    node server.js

## Docker
docker build -t my-eslint .
docker run my-eslint

## How to start
After running the project, open a browser and go to:

http://localhost:5173

The webpage should load and display the content from the project.
